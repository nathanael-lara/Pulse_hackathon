# VisitFlow iPhone

Mobile-first companion app for the VisitFlow concept, designed specifically for iPhone-sized screens.

## Run

```bash
cd visitflow-iphone
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

- This app is intentionally separate from `visitflow/`
- The UI is optimized around iPhone ergonomics:
  - stacked cards
  - bottom navigation
  - large tap targets
  - focused single-column information flow
